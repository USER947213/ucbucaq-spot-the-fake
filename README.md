### Краткое описание проекта
Интерактивная игра-симулятор на движке Unity (UI Toolkit), обучающая выявлению фишинговых угроз в сети.

**Реализованные ключевые механики:**
1. **Процедурная генерация уровней:** Динамическая сборка веб-страниц (Ozon, ГосУслуги, СберБанк) из случайных модулей хедера, тела и подвала с инжекцией случайного количества аномалий.
2. **Экранная лупа:** Инструмент криминалистического анализа, увеличивающий текст при наведении и подсвечивающий скрытые латинские омографы (английские буквы в русских словах) красным цветом.
3. **Совет старшего аналитика:** Ограниченный инструмент подсказок (2 заряда на игру), подсвечивающий случайную ненайденную уловку на странице яркой оранжевой рамкой на 3 секунды.
4. **Интерактивные окна угроз:** Реализованы визуальные шаблоны фальшивого чата поддержки и системного предупреждения об обнаружении вирусов, блокирующие экран до их обнаружения.
5. **Система очков:** Динамическое начисление очков (+100 за уловку, +250 за чистый сайт, штрафы за промахи) и бонус за скорость прохождения.
6. **Таблица рекордов (Топ-5):** Локальное постоянное сохранение рекордов в PlayerPrefs с возможностью ввода имени при новом рекорде.

### Требования к системе
* **Среда разработки:** Unity 6 LTS (версия 6000.3.19f1 или выше)
* **Платформа сборки:** WebGL, Windows / macOS standalone

### Как запустить проект в Unity
1. Склонируйте репозиторий на ваш компьютер.
2. Запустите Unity Hub и нажмите кнопку **Add -> Add project from disk**.
3. Выберите папку **`PointAndClickGame`** внутри репозитория и откройте её в редакторе **Unity 6**.
4. В панели `Project` перейдите в папку `Assets > Scenes` и дважды кликните по сцене **`Game`**.
5. Нажмите кнопку **Play** в верхней панели редактора для запуска симулятора.
